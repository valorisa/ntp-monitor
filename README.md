# NTP Monitor

Un outil de surveillance NTP qui permet de :
- Analyser la dérive d'horloge
- Surveiller les serveurs NTP
- Générer des alertes
- Visualiser les métriques via une interface web

## Installation

```bash
# Créer un environnement virtuel
python -m venv venv
source venv/bin/activate

# Installer les dépendances
pip install -r requirements.txt
```
